# Privacy Policy

Last updated: June 15, 2026

Simple row counter is an Android app for knitting and crochet row counters. The app does not run
its own server and does not sell user data.

## Data Stored By The App

The app stores row counters, project names, linked counters, goals, and local app settings on
the user's device. If the user imports a PDF pattern, that file is also kept on the device, in
the app's private storage.

If the user enables Google Drive sync, the app also stores the selected Google account email on
the device so it can show which account is connected.

If the user enables Google Drive sync, the app stores a sync snapshot in the user's Google Drive
app data folder. This folder is private to the app and is not visible in the user's normal Drive
files.

If the user starts a subscription or free trial, Google Play manages the purchase and payment.
The app checks the subscription status through Google Play Billing. The app does not receive or
store payment card details.

## Google Account Access

Google Drive sync is optional. The app asks for the Google Drive app data permission only when
the user turns sync on.

The app uses this permission only to read, create, update, and delete its own sync data in the
user's Google Drive app data folder. The app does not access the user's visible Drive files.

The app does not store Google passwords. Google Play Services manages the Google authorization
tokens.

## Use Of Google User Data

Google user data is used only to provide the user-facing Google Drive sync feature.

The app does not use Google user data for advertising, analytics, credit decisions, AI model
training, or any unrelated purpose.

The developer does not transfer, sell, or disclose Google user data to third parties. Google
Drive and Google Play process data only as the services the user chooses to use for sync and
billing.

The app's use and transfer of information received from Google APIs follows the Google API
Services User Data Policy, including the Limited Use requirements.

No developer server stores Google user data or Google authorization tokens.

## Data Sharing And Tracking

Simple row counter does not sell user data.

The app does not use ads, analytics, or tracking software.

This website is hosted with GitHub Pages and proxied through Cloudflare. It does not use analytics
or advertising cookies.

## Security

Local app data is stored in Android app-private storage. Google Drive sync data is stored in the
user's Google Drive app data folder. Network requests to Google Drive, Google Play, GitHub Pages,
and Cloudflare use HTTPS.

## Retention And Deletion

Users can delete local app data by deleting projects in the app or uninstalling the app.

Turning sync off in the app stops further syncing but leaves the existing sync snapshot in the
Google Drive app data folder. The developer cannot access that folder outside the app because the
app does not operate a server and does not store Google authorization tokens.

Users can remove the app's Google access from their Google Account settings. If Google offers an
option to delete associated app data during that flow, use it to remove the Drive app data snapshot.
